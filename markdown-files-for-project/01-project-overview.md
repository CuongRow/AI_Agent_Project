# Project Overview

## Project Name

Java Mastery Learning Platform

## Goal

Build a modern responsive learning platform that helps users learn:

1. Java Core
2. Object-Oriented Programming
3. Data Structures
4. Algorithms
5. Problem Solving with Java

The platform must provide:

- Learning materials
- Interactive quizzes
- Progress tracking
- User accounts
- Admin management
- Responsive UI

## Tech Stack

Frontend:
- React 19
- React Router
- Axios
- Tailwind CSS
- Shadcn UI

Backend:
- Spring Boot 3
- Spring Security
- JWT Authentication
- Spring Data JPA
- MySQL

Deployment:
- Docker
- Nginx
- GitHub Actions

Architecture:
- RESTful API

# Business Requirements

## Guest

Can:

- View homepage
- Browse course categories
- Search topics
- Read previews

Cannot:

- Take quizzes
- Save progress

## Student

Can:

- Register account
- Login
- Study lessons
- Take quizzes
- View progress
- Bookmark lessons

## Admin

Can:

- Manage users
- Manage lessons
- Manage quizzes
- Manage categories
- View statistics
